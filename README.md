# Nestjs_Project
Nestjs CRUD with Authentication.

## Functionality: 

<ul>
<li>User CRUD.</li>
<li>Products CRUD.</li>
<li>Database Relations.</li>
<li>Login + Signup.</li>
<li>Auth</li>
</ul>

## Stack:

<ul>
<li>Backend: REST API - NestJs.</li>
<li>Database: MYSQL - Typeorm</li>
<li>Frontend: ReactJs</li>
<li>Auth: JWT, Passport, Bycrypt.</li>
</ul>
